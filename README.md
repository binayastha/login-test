## About Project

Steps to setup

- run `composer install` to setup all required packages
- run `cp .env.example .env` to copy .env.example to .env file
- run `php artisan key:generate` to generate app key
- set up the database connection (this project uses mysql and database name is `testwork`)
- run `php artisan migrate` after setting up database connection
- run `npm install` and `npm run dev`
- run `php artisan serve` to run the project
